accept

(accept this)

Accept an incoming connection from a remote node.

close

(close this)

Close the node.

connect

(connect this peer)

Open a connection to a remote node.

get-cookie

(get-cookie this)

Get the authorization cookie used by this node.

get-hostname

(get-hostname this)

Get the hostname part of the nodename.

get-name

(get-name this)

Get the name of this node.

get-names

(get-names node)

An alias for clojang.jinterface.otp.nodes/get-names that returns a list of a node’s registered mailbox names as a list of strings.

get-pid

(get-pid this)

Get the Erlang PID that will be used as the sender id in all “anonymous” messages sent by this node.

new

(new & args)

An alias for clojang.jinterface.otp.nodes/node but one that allows for symbols and keywords to be used as node names, a closer match for BEAM language nodes, which use atoms for their names.

peer

(peer & args)

An alias for the constructor clojang.jinterface.otp.nodes/peer but one that allows for symbols and keywords to be used as node names, a closer match for BEAM language nodes, which use atoms for their names.

ping

(ping this-node other-node)(ping this-node other-node timeout)

An alias clojang.jinterface.otp.nodes/ping that also allows for a 2-arity call (with the default timeout set to 1000).

publish-port

(publish-port this)

Make public the information needed by remote nodes that may wish to connect to this one.

register-status-handler

(register-status-handler this handler)

Register interest in certain system events.

self

(self & args)

An alias for for the constructorclojang.jinterface.otp.nodes/self but one that allows for symbols and keywords to be used as node names, a closer match for BEAM language nodes, which use atoms for their names.

unpublish-port

(unpublish-port this)

Unregister the server node’s name and port number from the Erlang port mapper, thus preventing any new connections from remote nodes.

whereis

(whereis & args)

An alias for clojang.jinterface.otp.nodes/whereis that also allows for the mailbox name argument to be a symbol, keyword, or string.
